# Build
```
git clone https://github.com/SimonLiang2/MAT335.git
cd MAT335

# virtual environment
python3 -m venv .env

# linux/macOS
source .env/bin/activate

# windows (powershell)
.env\Scripts\Activate.ps1
# windows (cmd)
venv\Scripts\activate.bat

pip install pygame
# from MAT335

python src/main.py
```
